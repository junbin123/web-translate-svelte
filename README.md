# WebTranslate-网页翻译插件

## 命令行

```shell
yarn

# 执行
yarn dev
# 实时更新dist文件
# 将dist导入浏览器


yarn build
```

## 技术栈

- svelte
- rollup

## 功能

- 双语对照
- 译文背景色更换
- 中日英互译
- 使用彩云翻译 api
-
